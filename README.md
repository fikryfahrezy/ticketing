### Prerequisites

- Bun 1.2 or higher

### Quick Start

#### Local Development

```bash
git clone <repository-url>
cd <clone directory>
```

#### Create .env file for each projects

```bash
# cd front-end / back-end
cp .env.example .env
```

And adjust the environment variable value if needed

#### Install dependencies

```bash
# cd front-end / back-end
bun install
```

#### Run the app

```bash
# cd front-end / back-end
bun run dev
```
